圆角图片

使用 border-radius 属性来创建圆角图片:

>>>椭圆形图片: img { border-radius: 50%;} >>>我们使用 border 属性来创建缩略图。 img { border: 1px solid #ddd; border-radius: 4px; padding: 5px;} >>>我们使用 border 属性来创建缩略图。在图片外层添加一个链接。 a { display: inline-block; border: 1px solid #ddd; border-radius: 4px; padding: 5px; transition: 0.3s;} a:hover { box-shadow: 0 0 2px 1px rgba (0, 140, 186, 0.5);} >>>响应式图片会自动适配各种尺寸的屏幕。 img { max-width: 100%; height: auto; }
